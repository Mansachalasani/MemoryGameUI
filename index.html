MEMORY GAME

TEST YOUR MEMORY!

ABOUT :

Test your memory with this memory game. First level doesn't have any time limit. On the game board, there are always two identical images. Start the game by flipping a card. Then try to find another card that has the same image as the first. If you can't find a pair, the flipped cards will be flipped back with the face down. Try to remember these images as it becomes easier to find pairs the longer you play. When you find a pair they are remain open the board and when you find all the pairs in this memory, you have completed the level and moves to the final level.In the final level, a 20seconds of count down time starts as soon as you make the first flip and every flip you make gets counted.
